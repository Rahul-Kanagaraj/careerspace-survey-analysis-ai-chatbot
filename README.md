# 📊 CareerSpace Survey Analysis AI Chatbot

An end-to-end **data analytics + AI chatbot solution** that processes survey data, performs NLP analysis, and enables interactive insights through a conversational interface.

---

## 🚀 Project Overview

This project analyzes student survey data to uncover insights using:

- Data Engineering (AWS + PostgreSQL)
- Natural Language Processing (NLP)
- Power BI Dashboards
- AI Chatbot (LangChain + OpenAI)

It transforms raw survey responses into actionable insights and enables users to interact with data using a chatbot.


---

## ⚙️ Key Features

### 🔹 Data Pipeline (AWS)
- Automated ingestion of CSV survey files from S3  
- Data transformation into structured format  
- Storage in PostgreSQL (RDS)  
- Built using AWS Lambda  

---

### 🔹 NLP Analysis
- Sentiment & emotion analysis  
- Topic modeling (LDA)  
- TF-IDF keyword extraction (unigrams, bigrams, trigrams)  

---

### 🔹 AI Chatbot (Streamlit + LangChain)
- Chat with survey data in natural language  
- Automated insights generation  
- Uses Pandas DataFrame agent  

---

### 🔹 Dashboard (Power BI)
- KPI tracking (survey trends, sentiment distribution)  
- Topic-based insights  
- Student feedback visualization  

---

## 📊 KPIs & Insights Generated

- Sentiment Score Distribution  
- Emotion Classification Trends  
- Top Topics from Survey Responses  
- Key Student Pain Points  
- Suggested Improvements & Features
