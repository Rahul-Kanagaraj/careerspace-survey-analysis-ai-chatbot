# careerspace-survey-analysis-ai-chatbot
 End-to-end analysis of multi-year student engagement survey data using NLP, AWS, and an AI-powered chatbot to deliver automated, data-driven career service insights.

This project analyzes 3 years of university career service survey data using NLP techniques such as sentiment analysis, emotion detection, and topic modeling.
It implements an automated AWS-based data pipeline and an AI-powered chatbot to provide real-time, personalized insights for improving student career services.
